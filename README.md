# Coding Interview Template

Please clone this repo. We will use this during our coding exercise.

If, for any reason, you are uncomfortable cloning this repo, please use [create-next-app](https://nextjs.org/docs/app/api-reference/cli/create-next-app) instead.

## Requirements

- Use `create-next-app` to create a blogging site
- Data should be fetched from an API and displayed in a UI
- Users can create, read, update and search for posts

Focus on functionality over style.
